## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- NPM v9+
- Solana CLI (optional)
- 0.3+ SOL in wallet

### Installation
```bash
# Clone repository
git clone https://github.com/devaxisb/solana-copy-trading-bot.git
cd solana-copy-trading-bot-main

# Install dependencies
npm install

# Add wallets to track
echo "TARGET_WALLET_1" >> wallets.txt
echo "TARGET_WALLET_2" >> wallets.txt

# Start the bot
npm run start
